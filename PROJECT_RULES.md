# AI Smart Campus Portal

## Project Goal

Build a production-ready AI Smart Campus Portal.

This is NOT a college CRUD project.

The UI must feel like Apple + Linear + Stripe Dashboard + Notion.

Everything should be clean, premium and scalable.

- Next.js 16
- TypeScript

- shadcn/ui
- Framer Motion

Backend

- Next.js Route Handlers

- FastAPI
